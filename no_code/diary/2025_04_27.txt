В результате рассмотрения graphviz,
можно создавать только один граф с одинаковыми
именами вершин.
Можно рисовать рамочки вокруг вершин, но если 
будут связи между группами вершин, рамочки будут 
преодолеваться стрелками.

Примеры графов

digraph G {
    // Определяем вершины
    A;
    B;
    C;
    D;
    E;
    F;
    
    // Первый граф
    A -> B;
    B -> C;
    C -> A;
    
    // Второй граф
    D -> E;
    E -> F;
    F -> D;
    A -> D;
    // Можно добавить ребра между графами, если это необходимо
}


digraph G {
    // Первый граф
    subgraph cluster_0 {
        label = "Граф 1";
        A -> B;
        B -> C;
        C -> A;
    }
    
    // Второй граф
    subgraph cluster_1 {
        label = "Граф 2";
        D -> E;
        E -> F;
        F -> D;
    }
}

Вывод:
На текущий момент для рисования нескольких предложений можно использовать graphviz.
Не будут работать фильтры по предложениям, но хоть как-то можно будет работать.
